### ?? Project Overview

This project is a complete redesign of the ESAG (Escola Secundária Almeida Garrett) website, focusing on **digital accessibility**, **usability**, and **modern web standards**. Built as part of a university project for the "Interfaces e Usabilidade" course.

### ? Key Features

#### ?? **Accessibility (WCAG 2.2 AA Compliant)**
- **Keyboard navigation** throughout the entire website
- **Screen reader support** with proper ARIA labels
- **High contrast mode** and scalable text options
- **Focus management** with visible indicators
- **Error prevention** and user guidance
- **Semantic HTML** structure

#### ?? **Design Principles Applied**
- **Nielsen's 10 Usability Heuristics** - All systematically implemented
- **Norman's Design Principles** - Affordances, signifiers, mapping, and feedback
- **Mobile-first responsive design**
- **Modern UI/UX patterns**

#### ??? **Technical Implementation**
- **React 18+** with functional components and hooks
- **React Router** for client-side routing
- **Custom accessibility hooks** for state management
- **CSS Grid and Flexbox** for responsive layouts
- **Progressive enhancement** approach
